# AgriDirect — React Native App

Farm-to-consumer mobile app connecting buyers directly with local farmers.

## Features
- Buyer: browse products, cart, checkout, orders, profile, saved addresses
- Farmer: product management, orders, AI assistant (Krishi AI), earnings dashboard
- Delivery: order management, GPS tracking, earnings
- Firebase Phone OTP authentication
- Cloudinary image upload
- Redux Toolkit + Redux Persist state management

## Tech Stack
- React Native (Android + iOS)
- Firebase Auth (Phone OTP)
- Redux Toolkit + Redux Persist
- React Navigation
- Appium/WebdriverIO E2E tests

## Setup
```bash
npm install
npx react-native run-android
```
